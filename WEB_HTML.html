1. HTML

HyperText Markup Language의 약자로 웹을 만드는 대표 3가지(HTML, CSS, JavaScript) 언어 중 하나입니다.


1.1. tag

태그(tag)는 < (tag) > 같은 것을 부르는 이름입니다.

여는 태그와 닫는 태그를 구분하기 위해 닫는 태그에는 /를 붙입니다

사용법은

< H(1~6) > (text) < /H(1~6) > 입니다.


1.1.2. U

underline의 앞 글자만 따온 줄임말로 밑줄을 긋습니다.
사용법은
< U > (text) < /U > 입니다.
